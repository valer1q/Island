# Island — симуляция экосистемы «Остров»

Учебный проект-симуляция экосистемы на острове: остров представлен сеткой клеток, в каждой клетке могут находиться животные и растения. На каждом тике симуляции существа перемещаются, питаются, размножаются, растения растут, а статистика выводится в консоль.

## Запуск тестов

```bash
mvn test
```
