# Island — симуляция экосистемы «Остров»

Учебный проект-симуляция экосистемы на острове: остров представлен сеткой клеток, в каждой клетке могут находиться животные и растения. На каждом тике симуляции существа перемещаются, питаются, размножаются, растения растут, а статистика выводится в консоль.

Репозиторий: https://github.com/valer1q/Island

---

## Как использовать этот архив

1. Распакуйте архив.
2. Скопируйте файлы **в корень** вашего проекта Island:
   - `README.md` (или объедините с вашим)
   - `pom.xml`
   - папки `src/test/...` и `docs/test-results/...`
3. Приведите проект к Maven-структуре (если ещё не сделали):
   - код приложения должен быть в `src/main/java/...`
   - тесты — в `src/test/java/...`

---

## Запуск тестов

```bash
mvn test
```
